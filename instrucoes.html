🚀 Controle de Estoque - Sistema Completo

📋 Visão Geral

Sistema completo de controle de estoque com interface web moderna, desenvolvido em Python (backend) e HTML/CSS/JavaScript (frontend).

🎯 O que você consegue fazer?

✅ Gerenciar Produtos

Adicionar, editar e deletar produtos do estoque

📥 Entradas

Registrar compras e devoluções com motivos

📤 Saídas

Registrar vendas, perdas e outros movimentos

📊 Relatórios

Visualizar valor total e histórico completo

⏰ Histórico

Ver todas as movimentações com data e motivo

🌐 Interface Web

Acessar pelo navegador em qualquer dispositivo

🚀 Passo 1: Instalar Dependências

Execute o comando para instalar as bibliotecas necessárias:

pip install -r requirements.txt

Ou simplesmente clique duas vezes em iniciar.bat (Windows) que faz tudo automaticamente!

🎬 Passo 2: Iniciar o Servidor

Opção A: Clique duplo (Recomendado)

  • Windows: Clique 2x em iniciar.bat
  • Linux/Mac: Abra terminal e execute ./iniciar.sh

Opção B: Terminal

python app.py

Você verá a mensagem:

* Running on http://127.0.0.1:5000

🌐 Passo 3: Abrir a Interface

Acesse a interface pelo navegador usando o servidor Flask:

http://localhost:5000

Se estiver usando outro dispositivo na mesma rede, troque localhost pelo IP do computador:

http://192.168.x.x:5000

📱 Interface - Explicado

📊 Aba: Estoque (Inicial)

Visualiza todos os produtos em cards com informações rápidas.

Botão Ação
➕ Entrada Adicionar unidades ao estoque (compra, devolução)
➖ Saída Remover unidades do estoque (venda, perda)
✏️ Editar Mudar nome ou preço do produto
🗑️ Deletar Remover produto do sistema

🏷️ Aba: Produtos

Gerenciar produtos: adicionar novos e visualizar em tabela.

  • Preencha o formulário com: Código, Nome, Preço, Quantidade
  • Clique em "Adicionar Produto"
  • Produto aparecerá na tabela e nos cards do estoque

📝 Aba: Movimentações

Histórico completo de entradas e saídas.

  • Veja todas as operações com data, hora e motivo
  • Filtre por código de produto
  • Últimas 100 movimentações exibidas

📈 Aba: Relatório

Resumo do estoque e exportação.

  • Total de produtos diferentes
  • Valor total em estoque
  • Tabela detalhada de cada produto
  • Botão para exportar PDF

🎯 Exemplos Práticos

Exemplo 1: Registrar uma Compra

  1. Vá para "📊 Estoque"
  2. Encontre o produto "Mouse Logitech"
  3. Clique em "➕ Entrada"
  4. Digite 50 unidades
  5. Motivo: "Compra fornecedor"
  6. Clique em "Confirmar"

✓ Resultado: Quantidade do Mouse aumenta em 50, movimentação é registrada

Exemplo 2: Registrar uma Venda

  1. Vá para "📊 Estoque"
  2. Encontre o produto "Teclado Mecânico"
  3. Clique em "➖ Saída"
  4. Digite 5 unidades
  5. Motivo: "Venda"
  6. Clique em "Confirmar"

✓ Resultado: Quantidade do Teclado diminui em 5, movimentação é registrada

Exemplo 3: Adicionar Novo Produto

  1. Vá para "🏷️ Produtos"
  2. Preencha o formulário:
    • Código: P010
    • Nome: Monitor 32"
    • Preço: 1500.00
    • Quantidade: 3
  3. Clique em "Adicionar Produto"

✓ Resultado: Novo monitor aparece em todos os lugares

📊 Estrutura de Dados

Os dados são salvos em estoque.json com esta estrutura:

{ "produtos": [ { "codigo": "P001", "nome": "Mouse Logitech", "preco": 89.90, "quantidade": 70 } ], "movimentacoes": [ { "tipo": "entrada", "codigo_produto": "P001", "quantidade": 50, "motivo": "Compra", "data": "2026-06-21 14:30:00" } ] }

Arquivo Descrição

index.html Interface web (frontend)

requirements.txt Dependências Python

⚙️ API Endpoints (Técnico)

Se quiser integrar com outro sistema, use estes endpoints:

Método Endpoint O que faz
GET /api/produtos Lista todos os produtos
POST /api/produtos Adiciona novo produto
POST /api/entrada Registra entrada de estoque
POST /api/saida Registra saída de estoque
GET /api/movimentacoes Lista movimentações
GET /api/relatorio Gera relatório completo

❓ Dúvidas Comuns

P: Por quanto tempo o servidor fica rodando?

R: O servidor fica rodando enquanto você não fechar a janela/terminal. Feche a janela do terminal para parar.

P: Posso acessar em outros computadores?

R: Sim! Se alterar a linha app.run() em app.py para app.run(host='0.0.0.0'), qualquer computador na rede pode acessar.

P: Os dados são perdidos se fechar?

R: Não! Os dados são salvos em estoque.json, então não são perdidos.

P: Posso usar em múltiplos usuários?

R: Sim! Qualquer computador com acesso à URL pode usar simultaneamente.

P: Como faço backup dos dados?

R: Faça uma cópia do arquivo estoque.json. Pronto!

🆘 Troubleshooting

❌ "Erro ao conectar ao servidor"

Soluções:

  • Certifique-se de que app.py está rodando
  • Recarregue a página (F5)
  • Verifique se a porta 5000 está livre

❌ "ModuleNotFoundError: No module named 'flask'"

Solução: Instale as dependências

pip install -r requirements.txt

❌ "Dados não aparecem na interface"

Soluções:

  • Verifique se estoque.json existe
  • Abra o console do navegador (F12) para ver erros
  • Recarregue a página
